Food Donation Management System
Console-Based Java Application
________________________________________
1. Project Overview
This project is a console-based system that helps manage food donations between donors and recipients.
Admins can review registrations and approve donations.
Users can donate food or request help based on their role.
The system demonstrates core Java concepts including:
•	Classes and Objects
•	Encapsulation
•	Constructors
•	Interfaces (service logic separation)
•	Collections (ArrayList, HashMap)
•	Menu-driven navigation
•	Basic validations
________________________________________
2. Objective
To build a functional Java application that:
•	Allows registration of donors and recipients
•	Lets donors donate food
•	Allows recipients to maintain their needs
•	Enables admin approval and monitoring
•	Generates simple donation reports

6. How It Works
1.	Users register as Donor or Recipient.
2.	Admin verifies and approves accounts.
3.	Donors submit food donation requests.
4.	Admin approves donations and assigns them to recipients.
5.	Recipients view assigned donations and status.
6.	Admin updates delivery information and views reports.
